# CS:APP Labs Repository

这个仓库用于记录和保存《深入理解计算机系统》（Computer Systems: A Programmer's Perspective, 3rd Edition）配套实验（Labs）的通关历程。

## 📁 目录结构说明

每个 Lab 的文件夹内通常包含以下两部分：
- **Handout**: 官方提供的实验初始代码与工具。
- **Writeup**: 官方的实验指导 PDF，详细介绍了实验要求与规则。

## 🛠️ 开发环境
- **OS**: Ubuntu 22.04 LTS (x86_64)
- **Compiler**: gcc (Ubuntu 11.4.0)
